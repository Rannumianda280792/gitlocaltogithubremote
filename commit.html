jika file repo sudah ada di git dan file digithub

namun ada team yang merubah file yang sama pada baris yang sama di github, maka

  • cek status commit terakhir pada github di git dengan perintah fetch
  • fetch untuk cek commit terakhir pada github(remote) : git fetch
  • lalu ketik git status
  • ternyata ada percabangan di repo tersebut cek di graph nya, ternyata ada conflic
  • maka kita ambil dulu file yang di remote dengan perintah git pull
  • kita ke vcode untuk solving conflict (auto open), kita hapus perubahan yang kita ingin kan
  • lalu kita git status, git add . , git commit, dan git push untuk mengrim kembali ke remote atau github
